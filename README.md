 EC2 Nginx Website Deployment
 Project Overview

This project demonstrates how to deploy a static HTML website on an AWS EC2 instance using Nginx.

---

 Steps Performed

1. Created a simple HTML webpage
2. Launched an EC2 instance on AWS
3. Connected using SSH
4. Installed and configured Nginx
5. Deployed the HTML file to Nginx directory
6. Configured Security Group to allow HTTP traffic

---

 Output

